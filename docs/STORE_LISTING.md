## 4. Datenschutz (Privacy) & Berechtigungen

*(Kopiere diese englischen Begründungen in das Google Dashboard)*

- **Single Purpose Description:**
  "WebNote allows users to create sticky notes, highlight text, and draw directly on web pages. The extension aims to improve reading and research productivity by providing persistent, local annotations."
- **Permission Justifications:**
  - `storage`: "Required to save the user's notes, text highlights, drawings, and toolbar preferences locally on their device."
  - `activeTab`: "Required to inject the drawing canvas and note UI exclusively on the webpage the user is currently viewing."
  - `scripting`: "Required to execute the content scripts that provide the annotation and highlighting logic on the current page."
  - `contextMenus`: "Required to allow users to quickly create a note or highlight selected text via right-click."
  - `<all_urls>` (Host Permission): "Required because users need to be able to make notes and draw on virtually any website they visit for research or reading purposes."
